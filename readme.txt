== Description ==

Product Lister for Walmart, by CedCommerce, helps sellers to get ready to upload products data from their WooCommerce store to Walmart.com for connecting it with one of the biggest e-commerce industry players.

WooCommerce, a free E-Commerce plugin for WordPress, allows users to sell anything on the internet. Designed to handle all sorts of stores, it supports small, medium and big businesses alike. And to cater to these businesses successfully sell WooCommerce products on Walmart, CedCommerce offers
Walmart-Product-Lister. This powerful extension helps to get your simple product ready for walmart.


= Main Features of Free Version  =

* Category mapping with walmart category.
* Create and edit profile settings.
* Create walmart attributes value for products.
* Get ready WooCommerce simple products for Walmart.
* Easy and simple configuration.
* Easy to use and install
* Easy to configure

== Installation ==

= Automatic installation =

In the search field type " Product Lister for Walmart" and click Search Plugins. Once you've found our  'Product Lister for Walmart' you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking "Install Now".

= Manual installation =

The manual installation method involves downloading the plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

== Screenshots ==

1. “Walmart details” that allows merchant to fill the basic walmart configuration details which merchant gets from https://seller.walmart.com/.
2. “Return Location settings” the merchant needs to enter the address where product has to be returned if any return is generated for orders.
3. To upload a product on walmart.com, a walmart category must be assigned to it. For this the merchant needs to map his/her WooCommerce category with walmart category. You can map your WooCommerce category with walmart under “Category Mapping” section.
4. This tab will enable you to create profile and where you can set the common attributes value for product and then assign profile to product from “Manage Product” Section. While creating profile you can also map your attributes field with any other meta field. Using profile you can handle common attributes values easily and also remove the redundancy of data.
5. "Manage product" section, it helps to get ready your product for uploading on walmart.
6. Fields details required for mentioning mandatory  walmart details.
7. Get all features to compare between paid and free version.
